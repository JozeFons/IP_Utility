"======================================================================================================"
				Necessary steps before first launch!!!
"======================================================================================================"

1. Create the IP_list folder and inside create the IP.txt file and then the IP_list insert into C:\ folder in your local disk in this way --> C:\IP_list\IP.txt 

Note: From update ver.1.2 you can skip first step. Check update log below.

2. Define your own IPs in the IP.txt - Don't forget to write the IP addresses in proper format. See example below.

3. If you set up your IPs then launch the IP_Utility.exe from any folder on your PC.

4. Enjoy the script!!! =) 

"======================================================================================================"
				Nevyhnutne kroky pred prvym spustenim!!!
"======================================================================================================"

1. Vytvorte IP_list zlozku a vo vnutri vytvorte IP.txt subor a potom IP_list vlozte do C:\ na lokalnom disku vo formate --> C:\IP_list\IP.txt 

Poznamka: Od aktualizacie ver.1.2 mozete preskocit prvy krok. Pozrite log nizsie.

2. Zadefinuj svoje IP adresy v IP.txt - Nezabudni zachovat spravny format pri v pisovani adries do listu. Pozri priklad nizsie.

3. Ak si zadefinoval adresy, spusti IP_Utility.exe z lubolneho suboru na pocitaci.

4. Dufam, ze skript posluzi!!! =)
